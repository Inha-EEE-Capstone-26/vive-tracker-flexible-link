# Data

This repository does not include raw experiment logs or full row-level datasets.
It provides public-safe samples, schemas, manifests, checksums, expected outputs,
and derived result tables required to verify the reported paper artifacts.

## Included

- `sample/`: small CSV samples for smoke tests
- `processed_manifest/`: manifest, feature, and label metadata
- `schema/`: data dictionary and coordinate-frame notes
- `checksums/`: hashes for included source-derived artifacts

## Excluded

- full clean dataset CSV files
- raw tracker logs
- temporary experiment dumps
- local absolute paths
