# 2-Link Synthetic Source-Processed Inputs

These two CSV files are the source-processed inputs for `data/processed/clean_dataset_2link_v1/`.

- `synthetic_only_target_plan_5000.csv`: synthetic target plan used by the 2-link dataset builder.
- `synthetic_only_supervised_samples_5000.csv`: supervised synthetic-label samples used by the 2-link dataset builder.

They are included because the school-submission package is intended to support detailed reproduction and audit of the processed dataset.
