# Data Availability

This public repository does not include raw experiment logs or full row-level datasets.

It provides:

- public-safe sample CSV files for smoke tests
- processed manifests
- feature and label column lists
- schema and coordinate-frame notes
- expected metrics
- derived result tables
- selected final figures
- checksums for included artifacts

Full processed datasets can be distributed later only after a separate disclosure review. If released, they should be provided through a versioned release asset, DOI-backed archive, or controlled request path with checksums.

Raw device logs, local paths, private notes, cache folders, and failed experiment dumps are intentionally excluded.
